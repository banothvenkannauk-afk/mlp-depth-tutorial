# mlp-depth-tutorial
# How Deep Should a Neural Network Be?

This project explores how the depth of a Multilayer Perceptron (MLP) affects performance, focusing on underfitting, balanced learning, and overfitting.

## Author
Banoth Venkanna

## Dataset
Scikit-learn Diabetes dataset (regression)

## Models
- Model A: (8) → Underfitting
- Model B: (32, 16) → Best performance
- Model C: (128, 64, 32, 16) → Overfitting

## What this project shows
- Effect of network depth
- Bias-variance tradeoff
- Generalisation vs memorisation

## Files
- tutorial.pdf → full assignment
- notebook.ipynb → code
- figures/ → graphs
